Cálculo de Argila para Cilindro





As peças cerâmicas no torno seguem a base de um cilindro. Para calcular uma quantidade estimada de argila para a produção de uma peça, leve em conta o tamanho do cilindro necessário para criar o formato desejado. Informe os valores de altura e largura em centímetros. A espessura da parede é medida em milímetros, sendo comumente utilizados de 5 a 7 mm de espessura para peças médias.
